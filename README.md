# Smart Light Controlling Using Sound (IoT)

This project demonstrates a smart lighting system that can be controlled using sound (clap detection). The system detects clap patterns and toggles a light using a sound sensor and microcontroller.

## Project Overview
The system uses a KY-038 sound sensor to detect sound signals such as claps. The Arduino Uno processes these signals and controls a relay module that switches the light on or off. An ESP8266 WiFi module enables IoT connectivity for optional remote control.

## Hardware Components
- Arduino Uno
- KY-038 Sound Sensor
- ESP8266 WiFi Module
- Relay Module
- Breadboard
- Jumper wires
- Power supply

## Features
- Clap-based light control
- Hands-free home automation
- IoT connectivity
- Low-cost smart home solution

## Applications
- Smart homes
- Assistive technology
- Energy-efficient lighting
- IoT learning projects
